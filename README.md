# BunkBuddy 🎯 

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen?style=flat-for-the-badge)](https://ameersuhail799.github.io/BunkBuddy/)
[![Tech Stack](https://img.shields.io/badge/Stack-JS%20|%20Tailwind%20|%20HTML-blue?style=flat-for-the-badge)](#)

BunkBuddy is a feature-rich, student-centric productivity and attendance management application. It reduces academic stress by shifting from retroactive tracking to proactive mathematical forecasting, enabling students to plan recovery strategies for institutional attendance requirements.

## 🚀 Key Features

* **Real-time Attendance Calculator:** Instantly computes exact percentages based on total vs. attended sessions.
* **Attendance Forecasting & Simulator:** Allows students to simulate future attendance outcomes before skipping or attending upcoming classes.
* **Recovery Planner:** Mathematically calculates exactly how many consecutive classes a student must attend to cross target thresholds (e.g., 75% or 80%).
* **Multi-Page Dashboard Architecture:** Features specialized dedicated views for subject tracking, historical logs, and simulation utilities.

## 🛠️ Technical Implementation

* **Frontend Architecture:** Built using semantic HTML5 and vanilla JavaScript (ES6+) utilizing explicit client-side logic for real-time calculations.
* **Styling Ecosystem:** Implemented using a mobile-first, completely responsive design via Tailwind CSS.
* **Deployment Workflow:** Streamlined and distributed publicly using automated GitHub Pages deployment pipelines.

## 📂 Project Structure

* `/index.html` - Primary dashboard view and analytics interface.
* `/calculator.html` & `/simulator.html` - Specialized functional modules handling proactive predictive logic.
* `/js/` - Core JavaScript modules managing client-side evaluation, predictive mathematical formulas, and state persistence.
